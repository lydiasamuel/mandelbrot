# mandelbrot

A simple rendering of the mandelbrot set which was adapted from the O'Reilly Programming to colourise the output using a VGA rainbow palette.

Example Input: 

```cargo run --release mandel.png 320x200 -2.0,1.0 0.6,-0.4```
